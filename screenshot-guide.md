## üé® Screenshot Best Practices

### General Guidelines
1. **Resolution:** Use high-resolution screenshots (at least 1920x1080)
2. **Format:** PNG for screenshots, JPG for photos
3. **Annotations:** Use arrows, boxes, or highlights to draw attention
4. **Consistency:** Use same style/theme across all screenshots
5. **Privacy:** Blur any sensitive information (API keys, personal data)

### Tools for Screenshots
- **macOS:** Cmd+Shift+4 (area), Cmd+Shift+3 (full screen)
- **Windows:** Snipping Tool, Windows+Shift+S
- **Browser Extensions:** Awesome Screenshot, Nimbus Screenshot
- **Annotation:** Skitch, Annotate, or built-in markup tools

### Tools for Annotations
- **macOS:** Preview (built-in markup)
- **Online:** Photopea, Canva, Figma
- **Desktop:** GIMP, Photoshop, Affinity Photo

### Creating Comparisons
1. Use side-by-side layout
2. Add "Before" and "After" labels
3. Highlight key differences
4. Use consistent sizing

### Creating Diagrams
1. Use tools like Excalidraw, draw.io, or Figma
2. Keep color scheme consistent
3. Use clear labels
4. Make it scannable
